****************************
Journald receiver
****************************

.. meta::
      :description: Parses Journald events from the systemd journal.

The Splunk Distribution of the OpenTelemetry Collector supports the Journald receiver. Documentation is planned for a future release.

To find information about this component in the meantime, see :new-page:`Journald receiver <https://github.com/open-telemetry/opentelemetry-collector-contrib/tree/main/receiver/journaldreceiver>` on GitHub.
