****************************
StatsD receiver
****************************

.. meta::
      :description: Collects StatsD messages to generate metrics.

The Splunk Distribution of the OpenTelemetry Collector supports the StatsD receiver. Documentation is planned for a future release. 

To find information about this component in the meantime, see :new-page:`StatsD receiver <https://github.com/open-telemetry/opentelemetry-collector-contrib/tree/main/receiver/statsdreceiver>` on GitHub.
